This repository contains three packages for [[https://www.gnu.org/software/emacs/][GNU Emacs]] forked from
[[https://github.com/leanprover-community/lean4-mode][Lean4-Mode]].

- ~nael~ defines ~nael-mode~, a major mode for [[https://lean-lang.org][Lean]], and integrates it
  into Emacs' builtin facilities like [[https://www.gnu.org/software/emacs/manual/html_node/emacs/Abbrevs.html][Abbrev]] (abbreviations), [[https://www.gnu.org/software/emacs/manual/html_node/eglot/][Eglot]]
  (LSP) and [[https://www.gnu.org/software/emacs/manual/html_node/autotype/][Skeleton]] (templates).

- ~nael-lsp~ integrates ~nael-mode~ into [[https://github.com/emacs-lsp/lsp-mode][lsp-mode]].

- ~nael-markdown~ integrates ~nael-mode~ into [[https://jblevins.org/projects/markdown-mode/][Markdown Mode]].
